This is a bot that can search for a particular person in vc then connect and play a sound
to use it you just add a particular person in the code and the sound and run it
The bot will automaticaly connect and disconnect
Then there's command like >join which let's him join and >talk which make's him play a sound
to add people you will have to write: >addperson @NAME sound/yourfile(or you can write your full direction)
AND if you want to delete data you can just write: >removeperson @NAME
